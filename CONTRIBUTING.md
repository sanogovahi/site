# Directives de contribution

Merci de vouloir contribuer au projet Police Nationale ! 🚔

## Comment contribuer

1. **Fork** le repository
2. **Créer une branche** (`git checkout -b feature/ma-feature`)
3. **Commit** vos changements (`git commit -m 'feat: description'`)
4. **Push** vers la branche (`git push origin feature/ma-feature`)
5. **Ouvrir une Pull Request**

## Standards de code

- ✅ Utiliser ES6+
- ✅ Indentation: 2 espaces
- ✅ Noms en français pour les variables métier
- ✅ Commenter le code complexe
- ✅ Tester avant de soumettre

## Signaler un bug

1. Aller sur [Issues](https://github.com/sanogovahi/site/issues)
2. Cliquer \"New Issue\"
3. Décrire le bug avec:
   - Étapes pour reproduire
   - Résultat attendu vs réel
   - Screenshots si pertinent
   - Informations système

## Demander une feature

Utiliser le template \"Feature Request\" sur Issues

## Commit Convention

Suivre Conventional Commits:
- `feat:` nouvelle fonctionnalité
- `fix:` correction de bug
- `docs:` documentation
- `style:` formatage
- `refactor:` refonte sans nouveau comportement
- `perf:` optimisations
- `test:` tests
- `chore:` tâches de maintenance

Exemple: `feat(api): ajouter endpoint recherche commissariats`
